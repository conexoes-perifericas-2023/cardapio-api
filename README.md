# Cardapio API
Aplicação backend para listagem dos itens do cardápio digital.

## Instalação

```bash
$ npm install
```

## Execução da API

```bash
$ npm run start
```

# Testes

### Testes unitários
```bash
$ npm run test
```

### Testes e2e
```bash
$ npm run test:e2e
```
